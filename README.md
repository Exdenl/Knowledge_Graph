# 艾伦·图灵 领域知识图谱构建系统 

本项目为《知识图谱》课程的个人作业。系统实现了一个端到端的传统NLP流水线，通过从零搭建大规模百科网络爬虫、特征工程、机器学习序列标注模型，最终构建出关于“艾伦·图灵”及其关联领域的交互式关系网络。

**本项目完全独立实现，未调用任何现成的大模型 API。**

## ✨ 核心学术与工程亮点

*   **🌐 领域数据增强与防封禁爬虫**
    *   基于 Wikipedia API 构建的大规模垂直领域语料库，包含 10+ 个核心实体词条。
    *   内置工业级正则化数据清洗模块，并支持断点缓存续传。
*   **🤖 远程监督去噪与 CRF 序列标注**
    *   采用**远程监督**思想，利用多类别种子词典实现自动数据打标，解决人工标注成本高的问题。
    *   引入**Hold-out 交叉验证**（80/20划分），拒绝虚高过拟合，并打印真实的独立验证集 F1 分数与 CRF 内部马尔可夫转移矩阵。
    *   深入的**形态学特征工程**：不仅提取词性与上下文窗口，还引入了汉字前后缀特征，大幅提升OOV识别率。
*   **🧩 浅层句法组块与指代消解**
    *   摒弃简单的“双实体+单动词”提取，设计了**多级动态滑动窗口**，向左右捕获介词与助词，精准提取复杂连词。
    *   引入**距离惩罚**消除跨从句噪音，以及**零形回指补全**处理省略主语的自然语言短句。
*   **🔗 知识融合与本体映射**
    *   通过正则化执行实体边界清洗，利用硬编码别名库进行实体对齐。
    *   引入**基于正则的本体对齐**，将海量杂乱的自然语言动词规范化为标准图谱边。
    *   计算多重边的**置信度权重**，并收敛图谱。
*   **🎨 交互式动态力导向图谱**
    *   基于 PyVis 的深色系科研大屏。节点大小映射**度中心性**，边粗细映射**知识置信度**。

## 📂 目录结构说明

```text
Knowledge_Graph/
├── data/
│   ├── raw/                # 爬虫抓取并深度清洗后的原始语料库 (.txt)
│   └── processed/          # 经过抽取与融合后的结构化三元组与权重 (.json)
├── models/
│   └── crf_model.pkl       # 训练好的 CRF 机器学习模型序列化文件
├── output/
│   └── output_graph.html   # 最终生成的交互式可视化知识图谱大屏
├── src/
│   ├── crawler.py          # Wikipedia 爬虫与正则化数据清洗模块
│   ├── utils.py            # NLP 特征工程抽取模块（N-Gram与形态学）
│   ├── model_trainer.py    # CRF 模型构建、交叉验证与训练评估模块
│   ├── inference_engine.py # 句法组块抽取引擎（NER & 关系挖掘）
│   ├── knowledge_fusion.py # 实体消歧、本体关系映射与权重合并模块
│   └── visualizer.py       # PyVis 力导向图、物理引擎与视觉渲染模块
├── main.py                 # 全链路端到端自动化控制台入口
└── requirements.txt        # 项目环境依赖配置文件
```

## 🚀 快速开始

### 1. 环境准备
确保使用 Python 3.8+，建议在虚拟环境中执行以下命令安装依赖：
```bash
pip install -r requirements.txt
```
*(主要依赖包括 `wikipedia`, `jieba`, `sklearn-crfsuite`, `networkx`, `pyvis`)*

### 2. 运行流水线
一键运行全链路系统。整个过程包括：语料拉取 -> 模型训练与评估 -> 关系抽取 -> 图谱收敛 -> 前端渲染。
```bash
python main.py
```
*注：如果因网络问题无法连接 Wikipedia，系统支持断点缓存机制，您可以直接手动将语料 `.txt` 放入 `data/raw/` 目录下再次运行。*

### 3. 查看交互式图谱
运行成功后，终端将打印详细的算法评估报告。请使用浏览器（推荐 Chrome 或 Edge）打开项目目录下生成的：
```text
output/output_graph.html
```
您可以在页面左上角体验动态物理引擎参数调节。
