
# 艾伦·图灵 知识图谱构建系统

本项目为《知识图谱》课程作业。系统旨在通过端到端的自动化流程，从非结构化百科语料中构建关于艾伦·图灵及其相关领域的知识网络。

## 🌟 核心亮点
- **自动化数据采集**：集成 Wikipedia 官方 API，实现多维关联词条的广度爬取。
- **机器学习实体抽取**：摒弃简单的规则匹配，采用 **CRF（条件随机场）** 模型进行序列标注，支持传统机器学习的特征工程（词性、上下文窗口等）。
- **知识融合模块**：针对 NLP 抽取中的噪声问题，设计了基于别名映射和模糊匹配的**实体对齐**逻辑，有效清洗“Mathison”、“他”等实体碎片。
- **语义关系过滤**：通过关系停用词库与语义长度校验，提升三元组的知识密度，过滤“关于”、“诸如”等无意义连接。

## 📂 目录结构说明
```text
Knowledge_Graph/
├── data/
│   ├── raw/                # 爬虫抓取的原始语料 (.txt)
│   ├── processed/          # 经过抽取的结构化三元组 (.json)
├── models/
│   └── crf_model.pkl       # 训练好的 CRF 机器学习模型
├── output/
│   └── output_graph.html   # 生成的知识图谱
├── src/
│   ├── crawler.py          # Wikipedia 数据采集模块
│   ├── model_trainer.py    # CRF 模型特征工程与训练模块
│   ├── inference_engine.py # 知识抽取引擎（基于句法与模型预测）
│   ├── knowledge_fusion.py # 实体对齐、消歧与去噪模块
│   └── visualizer.py       # PyVis 动态交互渲染模块
├── main.py                 # 全链路自动化入口
└── requirements.txt        # 环境依赖配置文件
```

## 🚀 快速开始

### 1. 环境准备
确保使用 Python 3.8+，执行以下命令安装依赖：
```bash
pip install -r requirements.txt
```

### 2. 运行流水线
一键完成数据爬取、模型训练、知识抽取与图谱可视化：
```bash
python main.py
```

### 3. 查看结果
运行结束后，在浏览器中打开output目录下的 `output_graph.html` 即可查看交互式知识图谱。
## 🛠️ 技术原理
1. **特征提取**：CRF 模型考虑了词汇的词性（POS）、词长、以及前后窗口（N-Gram）的语法特征。
2. **远程监督**：通过已知种子实体库对百科文本进行自动标注，生成大规模训练样本。
3. **实体消歧**：基于上下文特征词匹配，区分“苹果（水果）”与“苹果（公司）”。
